# Teacher Quick Start Guide: Grading Short Answer Quizzes

## Overview
The new GradeBook allows you to inspect and grade quiz submissions question-by-question, especially for short answer and project questions that require manual grading.

## Step-by-Step Instructions

### 1. Access the GradeBook
```
Teacher Dashboard → GradeBook (in sidebar)
```

### 2. Select Assignment to Grade
- You'll see a dropdown with assignments that have pending submissions
- Each assignment shows how many submissions need grading
- Example: "test 1 - General (1 pending)"

### 3. View Submissions
After selecting an assignment, you'll see a list of student submissions:
- Student name and ID
- Submission date/time
- Current status (SUBMITTED or GRADED)
- Current score (if already graded)

### 4. Grade a Quiz Submission

#### Click "Inspect & Grade" Button
This opens a detailed grading dialog with two tabs:

#### Tab 1: Questions (Detailed Inspection)
For each question, you'll see:

**MCQ Questions (Auto-graded)**
- Question text
- Student's selected answer
- Correct/Incorrect badge
- Points earned (automatically calculated)
- Green "Auto-graded" indicator

**Short Answer Questions (Manual Grading)**
- Question text
- Student's text response (full answer visible)
- **Points input field**: Enter points earned (0 to max points)
- **Comment field**: Add feedback for this specific answer

**Project Questions (Manual Grading)**
- Question text
- List of uploaded files (clickable to download)
- **Points input field**: Enter points earned
- **Comment field**: Add feedback

#### Tab 2: Summary (Final Grading)
- **Calculated Score**: Automatically calculated from all answers
- **Max Score**: Total possible points
- **Final Score Override**: Optional - enter a different score if needed
- **Overall Feedback**: Add general comments for the entire quiz

### 5. Save the Grade
Click "Save Grade" button:
- Individual question grades are saved
- Overall submission is marked as GRADED
- Student can now see their results
- Submission list refreshes automatically

## Example Grading Scenario

### Quiz: "test 1" with 2 questions
1. **Question 1 (MCQ, 5 points)**: Auto-graded as Correct = 5 points
2. **Question 2 (Short Answer, 5 points)**: Needs manual grading

### Grading Process:
1. Open "Inspect & Grade"
2. Go to Questions tab
3. See Q1 is already graded (5/5 points)
4. For Q2, read student's answer
5. Enter points: 4 (out of 5)
6. Add comment: "Good answer, but missing key point about..."
7. Go to Summary tab
8. See calculated score: 9/10
9. Add overall feedback: "Great work overall!"
10. Click "Save Grade"

## Tips for Effective Grading

### 1. Use Question Comments
- Provide specific feedback per question
- Helps students understand what they got right/wrong
- More valuable than just a score

### 2. Review All Questions First
- Look through all answers before assigning points
- Ensures consistent grading across students
- Helps you gauge overall performance

### 3. Use Score Override Sparingly
- The calculated score is usually accurate
- Only override if there's a special circumstance
- Document reason in overall feedback

### 4. Provide Constructive Feedback
- Point out what was done well
- Explain what could be improved
- Give specific examples when possible

### 5. Grade in Batches
- Grade all submissions for one assignment at once
- Maintains consistency in grading standards
- More efficient than switching between assignments

## Common Questions

### Q: Can I edit a grade after saving?
A: Yes! Just click "Inspect & Grade" again on the same submission. Your previous grades will be loaded, and you can modify them.

### Q: What if a student deserves partial credit?
A: Use the points input field to give any value between 0 and max points. For example, if a question is worth 5 points, you can give 2.5 points.

### Q: Can I see what the student submitted for MCQ questions?
A: Yes! The Questions tab shows the selected answer for MCQ questions, along with whether it was correct or incorrect.

### Q: What happens if I don't grade all short answer questions?
A: The system will calculate the score based on what you've graded. Ungraded questions will count as 0 points.

### Q: Can I download student files?
A: Yes! For project questions, click on any uploaded file name to download it.

## Keyboard Shortcuts

- **Tab**: Move between input fields
- **Enter**: Save grade (when focused on Save button)
- **Esc**: Close grading dialog

## Troubleshooting

### Issue: Assignment doesn't appear in dropdown
**Solution**: The assignment may not have any submissions, or all submissions are already graded. Check the assignment list in "Teacher Assignments" page.

### Issue: Can't see student's answer
**Solution**: Make sure you're on the "Questions" tab. Scroll down if the answer is long.

### Issue: Score calculation seems wrong
**Solution**: Check that you've entered points for all short answer questions. MCQ questions are auto-graded and included automatically.

### Issue: Save button is disabled
**Solution**: Make sure you've entered points for at least one question, or entered a final score in the Summary tab.
